1.  Explain the differences between `client-side routing` and `server-side routing`.

Client-side routing is a setup how routes to different files and folders are implemented in browser.
Server-side routing is a setup how routes to data or database at the backend 

1.  What does HTTP stand for?

Hypertext Transfer Protocol

1.  What does CRUD stand for?

Create, Read, Update, Delete. These are basic actions to handle data.

1.  Which HTTP methods can be mapped to the CRUD acronym that we use when interfacing with APIs/Servers.

Create => POST
Read => GET
Update => PUT
Delete => DELETE
