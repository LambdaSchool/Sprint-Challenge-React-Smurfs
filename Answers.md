## Questions

1. Explain the differences between `client-side routing` and `server-side routing`.
1. What does HTTP stand for?
1. What does CRUD stand for?
1. Which HTTP methods can be mapped to the CRUD acronym that we use when interfacing with APIs/Servers.
1. Mention three tools we can use to make AJAX requests.

##Answers

1. Client-side-routing uses the client's web browser to route to different destinations of a website / single-page-application. Server-side-routing utilizes the server to receive routing destinations (server requests and new html to render).
1. HTTP stands for Hypertext Transfer Protocol.
1. CRUD stands for Create Read Update Delete.
1. Create is POST-like and Read is similar to GET via HTTP.
1. Axios, Redux, and XHR.
